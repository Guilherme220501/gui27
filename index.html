Atividades Extras curriculares


Desenvolvi
Cursadas
Revisão de Código
Análise e feedback sobre o código de outros desenvolvedores para melhorar a qualidade e identificar bugs.
Documentação
Criação e atualização de documentação técnica e de usuário para garantir que o software seja compreensível.
Testes Automatizados
mplementação de testes automatizados para garantir que o software funcione como esperado e para facilitar futuras modificações.
Refatoração
Melhoria do código existente sem alterar sua funcionalidade, visando aumentar a legibilidade e manutenção.
Análise de Requisitos
Colaboração com stakeholders para identificar e documentar as necessidades e expectativas do software.
Gestão de Configuração
Controle de versões e gerenciamento de alterações no código e na documentação.

Linguagens


Linguagens
Python
Linguagem versátil e de fácil leitura, amplamente usada em ciência de dados, desenvolvimento web e automação.
Java
Linguagem robusta e orientada a objetos, popular para desenvolvimento de aplicações empresariais e móveis (Android).
Javascript
Linguagem essencial para desenvolvimento web, permitindo interatividade e dinamismo nas páginas.
C
Linguagem de baixo nível, fundamental para sistemas operacionais e desenvolvimento de software próximo ao hardware.
C++
Extensão do C com suporte à programação orientada a objetos, usada em aplicações de alto desempenho e jogos.
C#
Linguagem desenvolvida pela Microsoft, popular para desenvolvimento de aplicações desktop e jogos (com Unity).
RUBY
Conhecida por sua simplicidade e produtividade, é amplamente usada no desenvolvimento web, especialmente com o framework Ruby on Rails.
